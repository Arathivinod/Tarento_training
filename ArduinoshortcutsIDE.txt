
Ctrl + N: Create a new sketch (Open a new tab)
Ctrl + O: Open an existing sketch
Ctrl + S: Save the current sketch
Ctrl + Shift + S: Save the current sketch with a different name
Ctrl + R: Verify/Compile the current sketch
Ctrl + U: Upload the current sketch to the Arduino board
Ctrl + Shift + U: Upload using Programmer (for boards with external programmer)
Ctrl + L: Go to the line specified by the line number
Ctrl + T: Auto format the sketch code
Ctrl + /: Comment/Uncomment the selected lines
Ctrl + W: Close the current sketch tab
Ctrl + Shift + T: Reopen the last closed tab
Ctrl + F: Find text within the current sketch
Ctrl + H: Replace text within the current sketch
Ctrl + Shift + F: Find and replace across multiple files
Ctrl + B: Go to the matching brace
